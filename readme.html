SLWiki How to use SearchEngine/PDF Viewer?

Use keywords words on search engine. Example: "EasyTouch user manaul" Note:As default it will bring user manual first or you can use "EasyTouch replacement parts" which will bring the direct link for PDF.

After you have found the pdf link, All you have to do is copy PDF link and paste it on PDFViewer so it can be viewed using Google Cloud Services except for downloading them.
