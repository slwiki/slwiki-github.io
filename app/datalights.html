SLWIki Data Lights

Solid Green | Orange fast blinks

No issue unless new install make sure the SL is plugged into LAN not WAN on router.

If ISP provider is Hughesnet, Only Local access will  work but NOT remote as well as If WAN router is being used, SL will not work locally or remotely due to security issue with network.

Solid Green | Orange 2 Blinks Pause 2 Blinks

Caused by: IP conflict with another device where SL is using same IP as another device. The issue occurs when the network equipment stops generating new IP's .

Fix: Power cycle the adaptor by taking Green connector out and leave it out for at least 30 seconds and then plug it back it should work fine OR use static IP for SL to highest IP that's available on network. except for DHCP

Permanent Fix: power cycle all network equipment.

Solid Green | Orange 2 Blinks, 1 Blink 3 Blinks

Caused by IP network changes such as new router or modem.

Fix: Power cycle the adaptor by taking Green connector out and leave it out for at least 30 seconds and then plug it back it might do the trick depending firmware version but more likely it will need HL device to reset IP manually.

Solid Green | Orange 4 Blinks, 1 Blink 4 Blinks

Caused by IP network conflict running through switch(s).

Fix: Plug the adaptor directly to modem or router except for going through switch.

Flashing  Green OR Orange ONLY.

Caused by bad power outage.

Fix: Needs brand new Ethernet cable used cabled might not work.

Solid Green | Orange with no pattern

Caused by new network |SL with old firmware.

Fix: More likely bad brick. HL and UDP might help.
