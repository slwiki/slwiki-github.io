• SL Compatibility

  • All EasyTouch
  • All Intellitouch
  • IntelliFlo VF, VS and VSF Only.

  • Routers

  • All Generic Routers that uses Local area network(LAN).
  • Not compatible with professional routers that uses Wide area network(WAN) which will run into OSI layer issue where IT professional will be needed if trying to use same network. The solution for professional routers is to have separate switch coming from Modem and going to ScreenLogic Adaptor along with their existing Network. Please note when doing this ScreenLogic will only have remote access unless Ethernet cable is plugged into modem or if the modem has Wi-Fi access as well which will work with either local or remote. The other solution is to have separate router coming from modem while existing router will remain same.
  • Not compatible with HughesNet satellite provider.
  • Some ISP Modems might have issue unless security is allowed manually.
